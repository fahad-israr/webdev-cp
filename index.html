Hi , this is Fahad Israr a Code golfer and an open source lover.


I’d like to convey my Visions for Web Dev and Competitive Programming Club of IIITT.


#What I’ve done so Far:

⇒ Establishing Helloworld: The Web Dev And Cp Club of IIITT.


⇒ Promoting the coding culture: by conducting regular coding competitions.


⇒ #Ha[.sh]: An open source global meetup organized by us to help freshers/noobs learn about open source.


⇒ Winter Of Code: An Open source program to improvise the dev culture and give opportunities to the freshers to work. We managed to bring Codechef as our sponsor(one of the biggest achievements)


⇒ Cypher Ball,The Code Party : IIITT’s first-ever global coding competition where even countries like North Korea participated


⇒ bit_by_bit : A “zero to hero program” launched to help people learn coding from scratch.


⇒ College Website: working with Anoop sir and other Juniors in the web Dev Club, I am developing newer website offering better UI, ergonomic Content Management and higher efficiency including features like PWA.


⇒ Virtual Learning Academy: Worked with Anoop Sir to build VLA portal and registration app as it was one of the acdemic projects of IIITT funded by MHRD.
